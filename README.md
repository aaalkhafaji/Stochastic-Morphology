# Stochastic-Morphology
The core numerical simulation of the forward and inverse transforms using discretized continuous scale-space operators.
tochastic Morphological Slope TransformThis repository contains the official numerical implementation of the Stochastic Morphological Slope Transform, a novel theoretical framework that establishes a strictly invertible non-linear spatial transform.Classical mathematical morphology suffers from idempotence (e.g., $\gamma \circ \gamma = \gamma$), meaning that operations like deterministic closings irreversibly destroy high-frequency structural information. This repository demonstrates how lifting an achromatic spatial signal into a $\pi$-mixture probability space bypasses this idempotence, allowing for exact expected reconstruction via the reverse transition measure $\pi^{-1}$.Theoretical FoundationUnlike the Fourier or Laplace transforms, which rely on linear $L^2$ isometries, this transform operates in the non-linear max-plus algebraic domain.The Forward Transform (Expected Supremum): Maps the spatial signal to a continuous geometric probability spectrum over a complete shape basis $B_\theta$.$$ \mathcal{M}_\pi[f](\theta) = \int_{\Omega} \left( \sup_{y \in \Omega} \{ f(y) + B_\theta(x-y) \} \right) d\pi(\theta) $$The Stochastic Inverse (Expected Infimum): Recovers the original spatial state $f(x)$ almost everywhere by integrating over the reverse Markov transition measure $\pi^{-1}$, effectively canceling the structural entropy of the forward diffusion.$$ \mathbb{E}[f(x)] = \int_{\Theta} \left( \inf_{y \in \Omega} \{ Y_\theta(y) - \tilde{B}_\theta(x-y) \} \right) d\pi^{-1}(\theta) $$
